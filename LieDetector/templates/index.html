Smart Lie Detector

Smart Lie Detector에 오신 것을 환영합니다!
당신의 거짓말 유무와 심리상태를 알려드립니다.

일반적인 거짓말 탐지기아니야..? 🤔

Smart Lie Detector는 다릅니다. 거짓말 탐지는 물론 정확한 결과값 출력을 위해 많은 센서를 통해 데이터를 뽑아내고,
RNN 알고리즘을 통해 다른 사람들의 데이터, 전에 검사한 기록들을 토대로 보다 정확한 심리상태와 불안상태, 생활패턴을
추천해 드립니다.

Icon

사용센서

- 피부 전기저항 센서 : 전기적 피부반사를 이용한 미세한 변화 측정
- 심박수 센서 : 사용자의 심박을 이용하여 미세한 변화 측정

Icon

MYSQL을 이용한 DB

데이터는 많을수록 정확한 결과값이 나올 확률이 높기에 사용자의 데이터를 확보하고 보다 정확한 결과값을
요구합니다.

Icon

Python LSTM
알고리즘

실시간으로 다수의 개인 데이터를 추출해 종합하여 평균값을 내고 효과적인 질문으로 머신러닝 을 통한 정확한 결과값을 도출합니다.
