히스토리🕒

1. 주제선정 완료.
2. 깃허브 개설.
3. 카카오 오븐을 이용한 기본 UI 프로토타입 설계.
4. 팀원 간 사용 프로그램 설치 및 통일화.
5. 아두이노 기본세팅.
6. 실제 UI HTML로 설계 및 디자인.
6. HTML을 플라스크를 이용한 파이썬과 연동.
7. 거짓말 탐지를 시각적으로 보여주는 그래프와 HTML UI와 연동.
8. 아두이노 센서값을 HTML로 받아오는데 성공.
9. Dbeaver를 이용한 파이썬 데이터베이스 구축.
10. 아두이노 센서값을 HTML 버튼 누를 시 그 값을 데이터베이스로 보내는데 성공.
11. RNN 알고리즘 구축.
12. 알고리즘에 필요한 실제 사용자 데이터를 추출.

사용된 프로그램💿

Python 3.7, Dbeaver 5.2.5, Flask, Arduino (Editor : PyCharm)
